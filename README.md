Aplicación full stack que permite el registro de usuarios, autenticación y 
administrar autos mediante las funciones CRUD(Crear, Leer, Actualizar y borrar)

Funcionalidades principales:
-Registro de usuarios.
-Autenticación basada en JWT.
-Crear, listar, eliminar o actualizar autos
-Protección de rutas tanto en frontend como en backend.
-Manejo de sesiones con jwt.

Tecnologías usadas:

Backend:

Java, Spring Boot, Spring JPA, Spring Security

Base de datos:

SQLServer

Frontend:

TypeScript, React, TailwindCSS
